.. zephyr:code-sample:: i2c-custom-target
   :name: I2C Custom Target
   :relevant-api: i2c_interface

   Barebones zephyr interface with HM01B0 using I2C and SPI. Transmits raw hex over UARTE.

Overview
********

Requirements:

nRF52832/52840: 1x
NPN BJT: 1x
4.7kΩ: 2x
100kΩ: 1x
Arducam HM01B0 camera: 1x
